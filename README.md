## Обо мне

Я системный администратор, которому важны порядок в инфраструктуре, автоматизация рутины и понятные инструменты для поддержки пользователей. Мне нравится превращать повторяющиеся ручные операции в предсказуемые сценарии, диагностические утилиты и рабочие процессы для support-команд.

Основные направления моей работы:

- автоматизация административных задач на PowerShell;
- Windows Server, Active Directory и Group Policy;
- DNS, DHCP, WDS и PXE;
- диагностика VPN, RDP и сетевых проблем;
- поддержка пользователей и Windows-рабочих станций;
- разработка внутренних инструментов для L2 и support-команд;
- развитие в сторону DevOps, CI/CD, Infrastructure as Code и Cloud.

## Проекты

### [Support Toolkit](https://github.com/AlexanderBabenko13/support-toolkit)

Набор PowerShell-утилит и скриптов для автоматизации задач технической поддержки и повседневной работы L2-инженера.

- диагностика сетевых, DNS и RDP-проблем;
- инструменты для работы с пользовательскими профилями и кешами;
- GUI-утилиты и интерактивные сценарии на PowerShell;
- автоматизация типовых L2-задач;
- подготовка portable-инструментов для внутреннего использования;
- сбор логов и экспорт диагностических отчетов.

`PowerShell` `Windows Support` `Diagnostics` `L2 Automation`

### [Secure Client Cleanup](https://github.com/AlexanderBabenko13/secure-client-cleanup)

WPF-инструмент для контролируемой очистки компонентов Cisco AnyConnect и Cisco Secure Client перед повторной установкой.

- безопасные действия по умолчанию;
- предварительный Scan и режим Dry Run / WhatIf;
- логирование, резервное копирование и экспорт отчетов;
- защита от случайного удаления широких системных путей;
- удобный рабочий процесс для инженеров поддержки.

`PowerShell 5.1` `WPF` `Safety-first` `Windows`
